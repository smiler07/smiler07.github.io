# smiler07.github.io

Personal professional website for GitHub Pages.

The Games section lets visitors choose between:

- Snake — responsive keyboard and mobile-touch game
- Shooting — browser-based JavaScript/Canvas game informed by the existing Python/Pygame prototype

- Repository: https://github.com/smiler07/smiler07.github.io
- Site: https://smiler07.github.io/
- Development loop: [AORR.md](./AORR.md)
- Current state and guardrails: [MEMORY.md](./MEMORY.md)
- Snake + Shooting execution prompt: [IMPLEMENTATION_PROMPT.md](./IMPLEMENTATION_PROMPT.md)
- Shooting prototype and Web port notes: [shooting_game/README.md](./shooting_game/README.md)

The `shooting_game/` Python source is a reference implementation. GitHub Pages must run the Web port without Python, Pygame, a backend, or the local virtual environment.
